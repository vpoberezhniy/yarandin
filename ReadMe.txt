Рекомендации по установке:

1. Скачайте с Github репозиторий:  git clone https://github.com/vpoberezhniy/yarandin.git
2. После клонирования выполните в консоле команду: composer install
3. Создайте новую базу данных (MySQL) с именем "yarandin"
4. В консоле выполните команду: php artisan migrate



!!! Важно !!!

Настройки файла .ENV Вы должны происать самостоятельно!
В даном проекте Вам необходимо прописать следующие настройки:
1) Подключение к БД (при написании проекта использовалось имя БД "yarandin")

Развертывание готово.

Как проверить фунции:

1) Для проверки функционала, на стартовой странице, нажмите на кнопку "Create new record"
2) Заполните форму (Добавьте несколько записей. Для проверки фильтра можно руками в базе изменить дату на разную)
